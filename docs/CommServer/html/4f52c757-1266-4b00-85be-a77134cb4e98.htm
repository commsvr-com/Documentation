APPENDIX A
Example of a report

Report contains following elements:

  1. Basic information - Report time, run time, configuration file and details about application and version;

  2. Stations - information about stations, read more about Station at runtime and during configuration

  3. Segments - information about segments, read more about Segments at runtime and during configuration

  4. Interfaces - information about ports (interfaces), read more about Ports at runtime and during configuration

  5. Protocols - information about protocols, read more about Protocols at runtime and during configuration
