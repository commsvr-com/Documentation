Trouble Shooting

Problem: I’ve unzipped all files (installed the application) successfully, but when I try to run it, an error occurs.

Solution: Please run CommServer as the Administrator for the first time. CommServer has to register itself as an OPC and Eventlog source.

Problem: While I am running CommServer my computer restarts suddenly.

Solution: CommServer implements a mechanism that protects against a deadlock. When CommServer finds that the deadlock occurs, it restarts the operating system. Please find a cause of the restart in the Eventlog. Improper configuration is usually the main cause of the deadlocks, so please verify the configuration or contact our technical support.
