X64 OPC Core Components installation

On 64 bit operating system there is necessity to install special version of OPC Core Components for x64 systems:

Figure 1: OPC Core Components for x64 systems

CommServer installation

The following version of CommServer should be installed: CommServerSetup_3_61_00_x86compilation.exe. This is special release based on CommServer 3.61.00 that is compiled to be run on 64 bit system as 32 bit application (in 32 bit compatibility mode).

This documentation is designed to be used with CAS CommServer (this software is available at http://www.commsvr.com).
