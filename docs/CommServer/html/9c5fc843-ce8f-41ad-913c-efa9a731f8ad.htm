Additional resources/tools

To simplify the configuration for a system that includes many remote objects NetworkConfig has built-in tools that allow to import some settings from different file types (Wizcon: *.bls, Excel: *.csv, etc.)

Figure 1: Import menu

The import tools:

  • WIZCON *.BLS files

  • Tag blocks

  • Tag bits

  • Tag mappings

  • Scan settings

This documentation is designed to be used with CAS CommServer (this software is available at http://www.commsvr.com).
