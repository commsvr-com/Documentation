DataPorter local service controller

Home

This tool strip that is available in DataPorter Monitor and OPC Viewer (DataPorter configuration) allows you to control the DataPorter local service. It allows for start, stop and restart of the local DataPorter (on the same PC as an application with that tool strip). This tool strip displays also the status of the local DataPorter service.

This documentation is designed to be used with CAS DataPorter (this software is available at http://www.commsvr.com).
