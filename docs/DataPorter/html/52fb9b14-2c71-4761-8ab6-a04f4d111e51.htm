DataPorter Standard Edition

This section contains the following subsections:

  • OPC bridging
  • OPC aggregation
  • OPC distribution
  • OPC calculation

DataPorter Standard Edition is OPC client software dedicated for flexible data transfer between OPC DA servers.

DataPorter Standard Edition is not only bridge between servers, it also allows to execute calculation and logical operations on OPC tags (see the configuration example).

DataPorter standard can execute the following operations:

OPC bridging

OPC aggregation

OPC distribution

OPC calculation
