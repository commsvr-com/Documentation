Licensing

Step 1. Download DEMO software

Download DEMO software available on website. The DEMO version is available to all. If you are a registered user you have access to the TRIAL version (Step 3).

Step 2. Install software

If you are interested in testing with more functional available freeware … go to the next step.

Step 3. Download the TRIAL license

The 14-day TRIAL license is available to registered users only. Register with our service to download it. If you need a longer TRIAL, register and apply for a longer individual license to be received via e-mail.

Step 4. Install the license

After starting the NetworkConfig software, you can install the downloaded TRIAL license in your target application in order to define required parameters.

Step 5. Order the end-user license

After having paid for the end-user license you will receive an individual end-user license certificate.

Step 7. Install the end-user license

After starting the NetoworkConfig software you can install the received end-user license in your target application following the procedure for TRIAL.

NOTE: The end-user license is related to a certain installation. It means that in case of software reinstalling your current certificate will become invalid. In that case you will receive a new license certificate free of charge, with the same terms and conditions as per your original license. In the temporary period of reinstallation you can use the TRIAL license, following the above procedure starting from step 3 (without step 6).

Licese Types

  • Commercial license (End User) – a payable license for an end user with defined parameters (max number of variables, term, runtime, functional constraints etc.) e.g. CS-SE-843, one PC workstation for CommServer Standard version without the term of use and runtime limitation, limited up to 843 process data variables (OPC tags). Commercial license is site constrained – it means that the license certificate is issued for a particular installation instance. It means that after reinstallation of software is necessary, it is impossible to apply the previous license certificate. In that case, the user receives a new license certificate free of charge with the same limits as the purchased license.

  • Academic license (APL) – a free of charge license for universities, colleges, academies, research centers and science institutes. The license has only a runtime limit.

  • DEMO license – a free of charge testing license for unregistered users. The license has all types of limits (max 50 tags, it expires after 14 days, and allows to keep software running for 2 hours).

  • TRAIL license – a temporary full access license for registered users and customers for testing, implementing and reinstalling. The license has only a term limit.

  • Special license (VAR) – an engineering license for integrators and vendors of the automation hardware and software. It has similar limits as the academic license (APL). It is issued for members of the partnership program. To obtain this license, an appropriate agreement has to be signed.
