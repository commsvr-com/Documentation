Home

I’ve installed the application successfully, but when I try to run it, an error occurs.

  • Please run DataPorter as the Administrator for the first time. DataPorter has to register itself as an OPC and Eventlog source.

When I try to run DataPorter with my own configuration only the error window appears instead of the main DataPorter window.

  • The reason could be a mistake in the configuration (e.g. in syntax of parameter for one of the transactions). Please verify the configuration and try to launch the application again. You can also try to launch DataPorter on the default configuration (if you do not have any default configuration please reinstall the application).

While I am running DataPorter my computer restarts suddenly.

  • DataPorter implements a mechanism that protects against a deadlock. When DataPorter finds that the deadlock occurs, it restarts the operating system. Please find the restart cause in the Eventlog. Improper configuration is usually the main cause of deadlocks, so please verify the configuration or contact our technical support.
