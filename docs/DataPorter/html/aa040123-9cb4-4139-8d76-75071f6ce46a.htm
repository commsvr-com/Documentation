DataPorter application contains of two components:

  • DataPorter engine – this component is responsible for OPC communication, as well as managing and execution of transactions and operations. See topic [TransactionAndOperations] to find more details.

  • DataPorter monitor – this component shows important information about DataPorter. User interface is based on tabs (see topics: [OPCClientTab], [GroupTab], [ItemTab], [TransactionTab]).
