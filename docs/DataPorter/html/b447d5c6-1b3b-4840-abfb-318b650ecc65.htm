ItemDescriber

Home

This tool allows us to generate a CSV file or an XML file that stores the configuration of a description for items configured in DataPorter or CommServer. The user can describe items by setting particular properties.

The main window is divided into two sections. On the right side, there is a list of properties and corresponding identifiers. On the left side, there are item names (syntax is “item name” or “server name/subscription name/item name”).

After adding an item name, click on “+” next to the name and select “ItmesItemProperty”. The list on the left changes the view to the property list for this particular item. The user can configure properties by adding couples: PropertyCode (code/identifier from the table on the right of the window) value of selected properties (Value).

 Note

When configuring properties remember to take OPC Foundation recommendations into consideration.
