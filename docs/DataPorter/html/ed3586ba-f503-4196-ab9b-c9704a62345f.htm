DataPorter has a built-in report utility that allows generating reports about the communication sate. All data provided by the tabs described previously are available in the report. The report is displayed in the Internet Explorer window and can be saved from this application for future analysis.

See topic: “Sample Report” to see a sample report.

To generate a report, select (in the DataPorter main window):

Menu -> Report

The report should be opened in the web browser window.

This documentation is designed to be used with CAS DataPorter (this software is available at http://www.commsvr.com).
