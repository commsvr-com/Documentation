Application menu

Access to additional modules and options is provided via the menu.

File menu
Figure 1: File menu

Please select:

  • Report – to display HTML report (see section: “Report utility” for additional details)

  • Exit – to close the CommServer Monitor application

View menu
Figure 2: View menu

  • Hide – to hide the main application window and minimize it to a tray icon

Tools menu
Figure 3: Tools menu

  • Configurator – to open NetworkConfig

  • DCOM Configuration – to start DCOM configurator (dcomcnfg) built-in Windows

  • XBUS Measure – to start DataProvider diagnostic application

  • Options – "CommServer Monitor" application options

  • Connection Type Options – to choose between the connection to CommServer or CommServer UA

Help menu
Figure 4: Help menu

  • Contents – to display help for the application

  • About – to display the "about" window
