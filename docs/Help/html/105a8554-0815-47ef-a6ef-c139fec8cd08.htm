Model 3D view

OPC UA Address Space Model Designer provides many views of the model. One of them is a 3D view (Figure 1). It is a three-dimensional tree view representation of the nearest environment of the node selected in the classic model view. To use the 3D view, the node in the Model view must be selected.

Figure: Model 3d View

Model view elements:

  1. Visualization panel - visualization of the three dimensional tree view representing the nearest environment of the node, selected in the Model view.

  2. 3D control panel - contains configuration parameters of 3D visualization: camera location. rotation etc.

  3. Visibility panel - enables/ disables lines, labels 2D/3D and triangle structures on the visualization panel.

  4. Axes scroll bars - sets the position of the 3D model in space (on X/Y/Z axes).

  5. Property panel - properties of the selected node.
