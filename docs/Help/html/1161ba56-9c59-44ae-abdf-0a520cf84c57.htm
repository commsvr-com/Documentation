Introduction

OPC UA Address Space Model Designer is not only a tool that can be used for the creation of the Information Model and for binding real-time process data, but it is also a tool that supports creating of the documentation. The "Model export" function can be useful to export particular nodes to the document in human-readable form. At present 2 types of export are supported:

  • Word (docx)

  • Maml

The created files can be used to build the documentation and help to understand the model structure.
