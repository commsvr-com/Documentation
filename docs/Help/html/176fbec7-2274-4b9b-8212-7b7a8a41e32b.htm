Instalation and First Launch

Home

This chapter guides you through the process of setting up CAS CommServer UA™. These instructions, which describe how to install CAS CommServer UA™, must be followed carefully to ensure a successful and trouble-free installation.

 Note

The installation of CAS CommServer UA™ has to be done on Administrator account.

Prerequisites

  • Microsoft . NET Framework 3.5

  • OPC Core Components

  • Windows 2000/Windows XP/Windows Vista/Windows 7

To install the application, proceed as follows:

  1. Launch Setup.exe or CommServerUASetup_X_XX_XX.exe (where x_xx_x is CAS CommServer UA™ version number). CAS CommServer UA™ Welcome dialog box appears.

    Read information and click Next.

  2. User information dialog box ( appears.

    Enter User name, Company name and e-mail address in appropriate text boxes and click Next.

  3. Installation dialog box appears.

    Default location for application files is specified as:

     Note

    C:\Program Files\CAS\CommServerUA

    You can change it to a different location by clicking Browse button or by writing the path manually in the text box. Disc Cost button shows the amount of necessary and available space on the target drive. Choose option Everyone to install CommServer for anyone who uses this computer or Just me option to install only for yourself.

  4. Confirmation dialog box appears.

    Click Next to start installation. You may also change any previous information by clicking Back button.

  5. Installation Progress dialog box appears.

    Installation in progress. Wait until your computer has finished the installation.

  6. Installation Complete dialog box appears.

    The installation has been successfully completed; click Close button to exit the installation.

    All necessary files have been copied to the target folder on your computer, the software components have been registered and short-cut icons have been created in the Start menu Start->Programs->CAS->CommServerUA.
