Transaction
  • - initial state (if the transaction has been created but DataPorter has never checked the condition whether the transaction can be executed)
  • - DataPorter at least once checked the condition whether the transaction can be executed; the other possibility is that the transaction has been executed but an error occurred afterwards
  • - the transaction has been executed successfully
Tag
  • - a tag has been created but it has not been added to the OPC server and has never been read
  • - quality is not good
  • - read operation is finished successfully, there is no problem with this item (tag)
  • - the last write or read operation failed, the item was not available in OPC server, or another error occurred
Group
  • - the group is not active
  • - there was a problem with some items(tags) that belong to that group
  • - there is no problem with this group
  • - the group cannot be added to the OPC server
Server
  • - the server is waiting to be connected
  • - the server is connected
  • - the server is disconnected
