Introduction

OPC UA Address Space Model Designer Professional is a powerful, advanced tool, which contains all features of the Standard edition and also many new features such as visualization of the OPC Unified Architecture Information Model as 2D/3D view and OPC Unified Architecture Address Space as a browse view, creation of the OPC UA server configuration with data bindings and much more.

Visualization

The OPC UA Address Space Model Designer Professional offers diverse views, flexible filtering and simulation functionalities to create a customized image of the model, best suited to the current tasks. These features not only increase the designing efficiency, but also improve robustness of the process by avoiding unexpected structural errors at the very early development stage.

The model can be visualized using the following modes:

  • Model view 2D - Classic 2D view of the Information Model (also available in Standard version)

  • Model view 3D - New three-dimensional view of the Information Model

  • Browse view 2D – The Address Space view that exposes the Information Model from the viewpoint of a client con-nected to the server

In 2D - OPC UA graphical notation is used, in 3D views - the designer can use scaling, rotating and change the view point to tailor the display to individual needs. A selected node, its properties and references can be edited using the property pane or custom editors.

Interoperability without programming

Architecture of the OPC UA Address Space Model Designer Professional allows vendors of the UA servers to develop plug-ins providing a configuration editor (in Standard edition only for CommServer UA). The editor is used to modify the general settings and define model nodes data-bindings with the real-time process. Advantages of this solution:

  • Simple configuration

  • No programming skills are required

  • An easier way to move your old OPC server to the Unified Architecture world

Import

The OPC UA Address Space Model Designer Professional allows designers to import the whole address space from an OPC DA server and create a new OPC UA Information Model. This new model can be used to create an OPC UA Wrapper and quickly migrate from the OPC DA world to the OPC UA world. Importing OPC DA address space is a straightforward method of deploying OPC UA servers and could be used as a migration path between the technologies.

Custom build tool

The OPC UA Address Space Model Designer Professional supports the custom built tool that may be used during the Information Model building (compiling) process. The default building tool that is delivered with ASMD is an OPC UA Foundation model compiler, however the user can develop and plug-in custom compiling tool.

Award

The OPC UA Address Space Model Designer is the winner in the “User interfaces” category of the “2009 Product of the Year” competition of Control Engineering Poland magazine. Our product has been praised by experienced industrial automation experts for innovation, user-friendliness, ease of use and possibilities of practical application.

Custom editors

The professional edition of ASMD supports new custom editors that provide special visualization for some node classes of the model. Those editors can be added as plug-ins to ASMD. Basically one custom editor is available in ASMD; it is a State Machine Editor for Finite State Machines.

Features

  • All OPC UA Address Space Model Designer Standard features

  • Model view 2D/3D visualization

  • Browse view 2D visualization

  • Export model documentation to .DOCX and .MAML

  • Import of the OPC DA server address space

  • Easy development of plug-ins providing a configuration editor for OPC UA servers

  • Custom editors

  • Custom built tool
