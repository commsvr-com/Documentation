How to connect remotely to CommServer UA with ADI model?

  1. Start OPC Foundation Sample UA Client and paste the following connection string: opc.tcp://commsvr.com:51234/UA/CAS_UA_Server – [None:None:Binary] into the text box as it is on the picture below and click "Connect" button

    Figure 1: OPC Foundation sample client window

  2. In "Server Configuration dialog configure the connection as is shown on the example window below:

    Figure 2: Server configuration

  3. In the "Open Session" dialog click OK button

    Figure 3: Open Session

  4. If the "Accept certificate" dialog appears (example below) - accept the certificate anyway.

    Figure 4: Accept certificate

  5. Now you can browse ADI aware server online

    Figure 5: Working OPC Foundation sample client
This documentation is designed to be used with CAS CommServer (this software is available at http://www.commsvr.com).
