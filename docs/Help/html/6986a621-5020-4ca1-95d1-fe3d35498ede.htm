Tag Bits

This import tool is designed to facilitate the configuration of tags that represent bits from other tags.

scr
Copy
BaseTagName;Bitnumber;Name
StanZasuw/K101;0;/X1
StanZasuw/K101;1;/X2
StanZasuw/K101;2;/X11

  • BaseTagName – name in the 'Tag definition' table

  • Bitnumber – indicates the specified bit in the register that should be used to calculate the value; it can be found in the 'Bits definition' table

  • Name – name in the 'Bits definition' table

This documentation is designed to be used with CAS CommServer (this software is available at http://www.commsvr.com).
