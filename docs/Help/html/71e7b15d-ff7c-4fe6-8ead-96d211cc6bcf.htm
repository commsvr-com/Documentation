CommServer UA components

The CommServer UA package consists of the components that are shown in the figure below.

Figure 1: CommServer UA configuration

The CommServer UA package can be divided into the following sets of features and functionalities:

  • Common parts and functionalities of CommServer UA

  • Modeling features of CommServer UA and OPC UA Address Space Model Designer

  • Redundant, Multi-Protocol, Multi-Channel Data Access Engine functionality

  • OPC Wrapper functionality

This documentation is designed to be used with CAS CommServer (this software is available at http://www.commsvr.com).
