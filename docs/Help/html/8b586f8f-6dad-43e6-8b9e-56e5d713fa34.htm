Protocol

Protocol – a communication protocol (a kind of language) which is used for communication with remote devices. An unlimited number of communication protocols can be selected for each channel.

Protocol configuration window is on a figure 2 (below):

Figure 1: Protocol window

To configure the DataProvider click the button with "...". The DataProvider configuration window, which appears, is on a figure 3. To accept the chosen protocol: click OK button.

Figure 2: DataProvider configuration window

Configuration of the particular protocols is described in those topics:

  • MBUS

  • SBUS RS

  • SBUS IP

  • MODBUS RS

  • MODBUS IP
