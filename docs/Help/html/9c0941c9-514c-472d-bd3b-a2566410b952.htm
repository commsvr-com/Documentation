WHY DO YOU NEED A PROFESSIONAL EDITION?

OPC UA Address Space Model Designer is available in two editions: Professional and Standard.

The Standard Edition contains basic modeling functions allowing a designer to create and edit the OPC Unified Architecture Information Models and easily publish them as the OPC UA Address Space. It is also integrated with the OPC UA online e-book.

The Professional edition additionally contains a variety of advanced features supporting Information Model and Address Space visualizations, documentation, import/export functions, the configuration of any OPC UA server and binding of the model nodes with process data. Its functionality may be easily expanded by adding a custom OPC UA server editor to manage model nodes process data bindings.

For the Professional editions, we have very powerful functions on our road map that will greatly improve and simplify the design process, including but not limited to: XML schema / UML import and export, Visio™ and Enterprise Architecture™ integration. Our work will also focused on the improvement of the user interface to make it intuitive.
