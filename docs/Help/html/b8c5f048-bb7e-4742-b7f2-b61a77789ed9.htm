DataPorter

Home

DataPorter is an independent package of the CommServer software family. It is designed to integrate industrial applications - as an engine to port data between most popular standards.

DataPorter is very flexible and can be used to:

  • Transport or move data between OPC Servers

  • Do some simple data processing (adding, subtracting, scaling, average calculation for selected tags),

  • Publish data using Web Services,

  • Buffer data in first in first out (FIFO) queue protecting important data,

  • Read or write from SQL databases.

DataPorter is like a bridge between technologies - it allows to interconnect applications implementing different technologies (standards) in order to manage data flow. Software is available in the following versions:

  • Standard – processed data transfer between OPC Servers

  • Professional – all options

This documentation is designed to be used with CAS CommServer (this software is available at http://www.commsvr.com).
