Menu UA Server

Below - the UA server menu with a description.

Figure 1: Menu UA Server

UA server menu items:

  1. Select the plug-in file, that contains an editor for the UA server. This is the only option that is available if no plug-in is selected.

  2. Remove the currently selected plug-in.

  3. Configuration options.

  4. Create a new configuration.

  5. Open the currently existing configuration, saved as a .uasconfig file.

  6. Save the currently opened configuration in the currently chosen location.

  7. Save the currently opened configuration in any location.

  8. Open the editor of the current configuration. See more in UA Server configuration
