Introduction to user interface

After starting of the application, the following window appears:

Figure 1: OPC UA Address Space Model Designer Main User Interface

Main user interface elements:

  1. Main panel - the user can view here the classic model view, model browse view or 3D model view.

  2. Help panel - the user can read here an intelligent e-book about the OPC UA technology, which is coupled with the OPC UA model.

  3. Model – a classic view of the model.

  4. Browse view - a browse view of the model (from the viewpoint of the server).

  5. Model 3D - a three-dimensional model view.

  6. Menu toolbar.

  7. Navigation toolbar - containing filters and navigation buttons.

  8. Search toolbar - user search for a particular node name in the model view.
