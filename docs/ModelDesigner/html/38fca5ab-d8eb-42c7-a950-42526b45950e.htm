Model verification in OPC UA Address Space Model Designer

The available model verification options are:

  1. Build of the solution – if the solution builds without any errors the model is correct. See: Model Build

  2. Browse View – a user can check if any error occurs during model build by browsing the model and investigating “error nodes”. See: Browse View tree
