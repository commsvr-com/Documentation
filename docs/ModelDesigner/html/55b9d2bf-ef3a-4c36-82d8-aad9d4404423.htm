Model import

Import of the model is the easiest way of creating an OPC UA Address Space from the existing model, OPC DA server address space or another source. At present OPC UA Address Space Model Designer supports import from OPC DA server address space. More information about this process can be found in the topic describing the DA import and in Walkthrough: Create UA Wrapper.
