Browse view

OPC UA Address Space Model Designer provides many views of the model. One of them is a browse view. In this view the model is shown from the viewpoint of the server. The browse view helps to understand the Information Model idea. The main purpose of using the browse view is to show how the OPC UA server "sees" the connections between model nodes, references and organization of the model structure. Using the browse view, the user can easily modify the classic model and check if the result comes up to expectations.
