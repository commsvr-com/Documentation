This section defines the node classes used to define nodes in the OPC Unified ArchitectureAddress Space. All node classes are derived from a common, Base node class.

The node classes defined to represent objects fall into three categories:

  1. Used to define instances

  2. Used to define types for those instances

  3. Used to define data types
