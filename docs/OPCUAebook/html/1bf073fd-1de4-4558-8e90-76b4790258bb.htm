Company overview

CAS provides real time and embedded solutions, including turnkey systems for monitoring/control and software development for real time distributed systems - software libraries of more than 1000000 source code lines used as building blocks in numerous applications. We use modern tools (mainly ORACLE and Microsoft frameworks) including high level languages, e.g. C#, C++, Modula-2, Java, Pascal (Delphi), Ada, etc., and operating systems: Windows, Linux and real-time Linux. We build open distributed applications using XML SQL, COM+, OPC, ODBC, etc. Safe operation of the systems is guaranteed by technologies such as VLAN, VPN, public key infrastructure, etc.

Due to the wide range of technology areas, CAS is familiar with and is able to develop unique total open systems for various applications. All of our major projects have been developed from scratch and have been working reliably for years.

Primary Markets

We have successfully completed a variety of highly complex projects, including a flight inspection system for ground radio navigation aids; distributed control systems of district heating networks, a distributed system for pollution emission monitoring for power plants; a telemetric system designed for remote monitoring and control of an unmanned pumping station for a power plant group; a computer-aided control system for one of the largest brown coal mines in the world; computer system integration for large-format visualization of information for industrial networks.

Main products and services

  • CommServer

    CommServer is a package of communication software to manage data transfer. Built-in technologies and algorithms provide an intelligent data transmission automatically adapting its parameters to the user’s or process needs.

  • CommServer UA

    CommServer UA is a package of communication software to manage data transfer. It’s a fully configurable OPC UA server provides a multi-protocol, multi-medium and multi-channel redundant access to physical process control device data. To ensure short response time and effective utilization of the physical device communication channels throughput, a unique scan on demand algorithm was implemented.

  • OPC UA Wrapper

    OPC UA Wrapper is an OPC UA server that wraps the Classic OPC DA server (DCOM based) and creates the Address Space based on the XML model file obtained from CAS Address Space Model Designer

  • OPC UA Address Space Model Designer

    UA Model Designer is a powerful tool, which allows creating, displaying and editing OPC UA models compatible with the OPC Unified Architecture address space concept. The tool is created for developers and engineers, for professionals and beginners.

  • DataPorter

    DataPorter is an independent package of the CommServer software family. It is designed to integrate industrial applications - as an engine to port data between most popular standards.
