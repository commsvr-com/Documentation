The OPC Unified Architecture server architecture models the server endpoint of client/server interactions. Figure 1 illustrates the major elements of the OPC Unified Architecture server and how they relate to each other.

Figure 1: OPC UA Server Architecture

The OPC Unified Architecture server application - a software package - is the code that implements the function of the server. It uses the OPC Unified Architecture server API to send and receive OPC Unified Architecture messages from OPC Unified Architecture clients. Note that the “OPC UA Server API” is an internal interface that isolates the server application code from an OPC Unified Architecture communication stack.
